# 開発ログ

**最終更新**: YYYY-MM-DD

開発作業の記録を格納するフォルダです。

## ファイル構成

```
dev-logs/
├── README.md                      # 本ファイル
└── YYYY-MM-DD_task-name.md        # 各作業のログファイル
```

## 命名規則

`YYYY-MM-DD_task-description.md`

- 日付は `date +%Y-%m-%d` コマンドで取得（手動入力禁止）
- task-description は英語またはローマ字で簡潔に

## ログファイルの書き方

CLAUDE.md の「開発ログガイドライン」を参照。
