## 運用ルール

1. **新しいテーブルを作成したら**
   - この対応表に追加
   - 該当ドメインの仕様書に詳細を記載

2. **どのドメインか迷ったら**
   - 最も関連が深いドメインに入れる
   - 複数にまたがる場合は主ドメインに入れて備考に記載

3. **新しいドメインを作成する場合**
   - `domain-template.md` をコピーして作成
   - この対応表のドメイン一覧に追加
